# minigames-design-docs

Este repositorio contiene el diseño, documentación y conceptos del proyecto **I'm a Little Dragon**, un juego cuqui de fantasía con minijuegos, exploración y una historia ligera inspirada en la relación entre un dragón bebé y una princesa.

> Posiblemente tambien haga algunos otros minijuegos o mecanicas que no tengan que ver, pero lo intentaria poner en otro repositorio
> El jego como tal, no esta desarrollado todavia (veremos como va), lo pondria tambien en otro repositorio

El objetivo del repositorio es:
- Organizar ideas de minijuegos + estética
- Definir mecánicas principales
- Crear una base para futuras implementaciones
- Servir como biblioteca de mecánicas reutilizables (Por ahora, no lo hare)

## 📚 Wiki
La documentación completa está organizada en la Wiki del repositorio.

- Relación con la historia (Si tiene)
